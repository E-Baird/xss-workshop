Resources

Things that might help you when developing XSS attacks in CTFs

All of the links below are useful resources when trying to launch a cross-site scripting attack in a CTF.

  • Tom Scott's XSS video for Computerphile: A basic overview of cross-site scripting and how it works. An oldie, but a goodie.
  • PayloadsAllTheThings's XSS Page: lots of example payloads and Javascript snippets for you to try, including smoke tests and cookie stealers
  • RequestBin: A site that gives you a custom URL and lets you view all HTTP requests made to that address
  • Portswigger's Guide to XSS: An XSS guide by the guys who make Burpsuite, another incredibly useful pentesting tool for intercepting, reading, and writing HTTP requests
